## Project Overview
Expense Tracker is a simple web-based application that helps users track their income and expenses.  
The project is designed with a clean user interface and focuses on basic expense management features.  
It is developed using only frontend technologies and works without any backend server.

The application contains two main pages:
- Expense Dashboard
- Add New Transaction



## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage
- Vercel (for deployment)



## Features

### Admin
- View all transactions
- Monitor total income, total expense, and balance
- Delete any transaction
- Manage overall expense data

### Teacher
- View transaction records
- Analyze income and expense summaries
- Read-only access to data

### Student
- Add new income and expense transactions
- View transaction history
- Filter transactions by category
- Track balance automatically



## How to Run Project
1. Download or clone the project files
2. Open the project folder
3. Run the project using:
   - VS Code Live Server extension  
   - OR open `index.html` directly in a browser
4. No backend or database configuration required



## Firebase Usage
Firebase is **not used** in this project.  
All data is stored locally using the browser's Local Storage, making the project lightweight and easy to understand for beginners.



## Conclusion
This project demonstrates how an expense tracking system can be built using only HTML, CSS, and JavaScript.  
It is suitable for learning frontend development, JavaScript logic, and local data storage concepts.
