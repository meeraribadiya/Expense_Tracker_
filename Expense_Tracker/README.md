# Expense Tracker

A simple and modern Expense Tracker application to help you manage your personal finances. Keep track of your income and expenses, visualize your balance, and filter transactions by category.

## Features

-   **Dashboard Overview**: View total income, total expense, and current balance at a glance.
-   **Transaction Management**: Add new income or expense transactions with date, description, category, and amount.
-   **Local Storage**: Your data is saved locally in your browser, so it persists even after you close the tab.
-   **Filtering**: Filter transactions by category (Food, Travel, Shopping, Bills, etc.) to analyze your spending.
-   **Responsive Design**: Works seamlessly on both desktop and mobile devices.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Modern styling with CSS Variables, Flexbox, and Grid.
-   **JavaScript (ES6+)**: Dynamic logic and DOM manipulation.
-   **LocalStorage API**: For data persistence.

## Setup

1.  Clone the repository or download the source code.
2.  Open `index.html` in your web browser.
3.  Start adding your transactions!

## Usage

1.  **Add Transaction**: Click the "+ Add Transaction" button. Fill in the details (Date, Description, Category, Amount, Type) and click "Save".
2.  **View History**: Your recent transactions will appear in the list on the dashboard.
3.  **Delete**: Click the trash icon next to a transaction to remove it.
4.  **Filter**: Use the dropdown menu to filter transactions by a specific category.
